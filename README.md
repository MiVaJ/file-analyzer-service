# File Analyzer Service

Веб-сервис для загрузки текстовых файлов из внешнего каталога и анализа частоты встречаемости цифр в их содержимом.

Приложение получает список файлов через внешний API с ограничением скорости запросов, выполняет скачивание, сохраняет информацию о загруженных файлах и позволяет рассчитывать статистику по выбранным файлам через веб-интерфейс.

Source API: `http://91.199.149.128:18001` ([Swagger](http://91.199.149.128:18001/docs))

## Возможности

- получение файлов из внешнего каталога с учётом ограничения количества запросов;
- пакетное скачивание файлов;
- предотвращение повторной обработки одинаковых файлов;
- отслеживание прогресса скачивания;
- сохранение информации о скачанных файлах в базе данных;
- просмотр списка скачанных файлов;
- пагинация списка файлов;
- расчёт статистики частоты встречаемости цифр:
  - общая статистика по выбранным файлам;
  - статистика для каждого файла отдельно;
- веб-интерфейс для запуска скачивания и анализа данных.

## Архитектура

Приложение построено как асинхронный backend-сервис с фоновой обработкой задач.

Основные компоненты:

- **FastAPI** - API и серверный рендеринг веб-страниц;
- **SQLAlchemy Async + PostgreSQL** - хранение информации о скачанных файлах;
- **Redis** - хранение состояния операций и прогресса скачивания;
- **Celery + RabbitMQ** - выполнение фоновых задач;
- **Nginx** - reverse proxy;
- **Docker Compose** - запуск приложения и инфраструктуры.

## Основные возможности приложения

### Скачивание файлов

Пользователь запускает процесс загрузки файлов через веб-интерфейс.

Сервис:

- получает имена файлов из внешнего каталога;
- учитывает ограничения API;
- скачивает файлы пакетами;
- сохраняет информацию о загруженных файлах;
- обновляет прогресс выполнения операции.

### Просмотр файлов

Раздел скачанных файлов позволяет:

- просматривать список загруженных файлов;
- использовать пагинацию;
- выбирать файлы для расчёта статистики.

### Расчёт статистики

Для выбранных файлов выполняется подсчёт количества каждой цифры в содержимом.

Результат содержит:

- общую статистику по всем выбранным файлам;
- статистику отдельно по каждому файлу.

## API

### Запуск скачивания

```
POST /api/download/start
```

Запускает фоновую задачу загрузки файлов.

---

### Получение прогресса скачивания

```
GET /api/download/progress/{download_id}
```

Возвращает текущее состояние операции скачивания.

---

### Получение списка файлов

```
GET /api/files
```

Возвращает список скачанных файлов с поддержкой пагинации.

---

### Расчёт статистики

```
POST /api/stats/compute
```

Принимает список идентификаторов файлов и возвращает статистику частоты встречаемости цифр.

Пример запроса:

```json
{
  "file_ids": [1, 2, 3]
}
```

Пример ответа:

```json
{
  "total": {
    "0": 467,
    "1": 478
  },
  "files": [
    {
      "filename": "example.txt",
      "digits": {
        "0": 54,
        "1": 41
      }
    }
  ]
}
```
